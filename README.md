# ExtractingBodyFromFile

Консольное приложение на C# для извлечения поля `xmlData` из свойства `body` внутри JSON-файла.

## О проекте

Этот проект создан для обработки JSON-файлов, в которых поле `body` содержит вложенный JSON в виде строки.

Программа:
- читает входной файл `sed_log.json` из папки `InputFile`
- обрабатывает массив объектов
- у каждого объекта берет значение поля `body`
- разбирает содержимое `body` как JSON
- извлекает из него поле `xmlData`
- формирует новый JSON-файл только с нужными данными
- сохраняет результат в папку `OutputFile`

Это удобно, когда в логах, экспортированных данных или промежуточных файлах полезная информация находится внутри строкового JSON, и нужно быстро извлечь только одно конкретное поле.

---

## Основная задача

На входе есть JSON-файл примерно такого вида:

```json
[
  {
    "id": 16,
    "url": "https://example.com/Upload",
    "body": "{\"xmlData\":\"PD94bWwgdmVyc2lvbj0iMS4w...\"}",
    "sended_at": "2026-02-27 15:56:11",
    "response_code": 0,
    "response": "",
    "type": "before"
  },
  {
    "id": 18,
    "url": "https://example.com/Upload",
    "body": "{\"xmlData\":\"PD94bWwgdmVyc2lvbj0iMS4w...\"}",
    "sended_at": "2026-02-27 15:56:17",
    "response_code": 0,
    "response": "",
    "type": "before"
  }
]
```
